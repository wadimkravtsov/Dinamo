Создание веб-сайта: «Коттеджные поселки города Барнаула»

(вторичный рынок загородной недвижимости).

Проект сайта включает в себя два приложения со страницами:



страница Villaqes с карточками коттеджных поселков с основной информацией по каждому: название, фото, удаленность от города, локация, пользовательский рейтинг. На боковой панели страницы - список риэлторов, которые занимаются продажей коттеджей. Из  карточки - переход на страницу с конкретными характеристиками поселка - фото с видами, инфраструктура, инженерные сети, общественный транспорт, контакты с администрацией поселка, средняя цена земельного участка, а также кнопки с переходом на страницы риэлторов, занятых продажей домов в данном поселке.  Комментарии пользователей.


страница Realtors с карточками риэлторов (Агентств недвижимости), занимающихся продажей домов и земельных участков в поселках, при клике по конкретному риэлтору - переход на страницу с его параметрами - адрес, контакты, ссылка на прайс - лист. Кнопки с названиями коттеджных поселков, в которых есть об’екты на продажу у данного риэлтора.
